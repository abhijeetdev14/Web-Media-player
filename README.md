🎵 Web Media Player
A lightweight and responsive web-based media player built using pure HTML, CSS, and JavaScript — no frameworks, no dependencies.
Supports audio/video playback, custom controls, progress tracking, and volume adjustments with a sleek, modern UI.

🚀 Features
🎶 Audio & Video Support — Play/pause and seek media easily

🔊 Volume Control — Adjust volume or mute instantly

📏 Progress Bar — Real-time tracking of playback position

📱 Responsive Design — Works on desktop and mobile

🎨 Customizable UI — Style with your own CSS

⚡ No Dependencies — 100% vanilla JavaScript
